# 🧠 NLP Transformers Journey

An interactive educational tool built with Streamlit to explain the step-by-step pipeline of processing natural language—from raw text to deep contextual embeddings.

## 🌟 Features
- **Interactive Cleaning & Normalization**: Dynamically toggle preprocessing steps and see real-time impact.
- **Visual Tokenization**: Explore word, subword, and character level splits.
- **Stopword & Stemming Visualization**: Understand how reduction helps models focus on meaning.
- **Dynamic Vocabulary Mapping**: See how words are indexed and their frequencies.
- **Mathematical Vectorization**: Compare Bag-of-Words and TF-IDF representations.
- **Spatial Word Embeddings**: 2D Visualization of semantic relationships using PCA.
- **Advanced Contextual Embeddings**: Witness how word meanings shift based on context (BERT-style).

## 🛠️ Tech Stack
- **Python**: 3.10+
- **Streamlit**: Elegant Web UI
- **NLTK / scikit-learn**: Core NLP processing
- **Plotly**: Dynamic visualizations
- **Transformers / sentence-transformers**: Deep semantic embeddings

## 🚀 How to Run
1. Ensure you have Python installed.
2. Setup a virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the app:
   ```bash
   streamlit run app.py
   ```

## 🎯 Educational Goal
Perfect for beginners and non-technical stakeholders to understand the core concepts of NLP without needing to write a single line of code.
